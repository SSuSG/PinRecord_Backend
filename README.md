# PinRecord

---
## 🗓 프로젝트 진행 기간
`2023.05.10 ~ 2023.05.24 (약 2주)`

---
## 📑 소개
오픈 API를 이용한 여행 기록 및 공유 SNS

---
## 🔑 제공 서비스
- Kakao Map API를 통한 여행지 검색 제공
- 다녀온 여행지를 KaKao Map에 핀으로 표시 및 클릭시 저장한 이미지 , 글 , 태그 등 정보 제공
- 태그/장소 기반 게시글 검색 제공
- 최신순 / 찜순 / 댓글순 정렬 기능 제공
- 게시글 작성시 팔로워에게 멘션 기능 제공
- 무한 스크롤 기능 제공
- 팔로우 / 추천 / 댓글 / 찜 등 SNS 기본 기능 제공
