<계정>
#회원가입
-> 이메일,패스워드,패스워드 확인 ,닉네임, 이름
-> 닉네임은 중복이 불가능하다(중복체크 기능)
-> 비밀번호는 8자 이상 영문,숫자,특수문자 1글자 이상
-> 이메일은 중복되면 안된다.(중복체크) 
-> 회원가입 성공시 인증이메일이 발송된다. 인증을 해야 로그인이 가능하다.#

#로그인
-> 이메일,패스워드로 로그인
-> 첫 로그인시 이메일로 인증하지않으면 로그인이 불가능하다.#
-> 비밀번호 5회 틀릴시 계정잠금#
-> 잠금을 해제하려면 이메일로 발송된 인증번호를 입력해야한다.#
-> 인증번호를 성공적으로 입력시 임시비밀번호를 발송해준다.#

#비밀번호 찾기
-> 비밀번호 찾기는 이름,이메일 입력시 일치하는 이메일로 임시 비밀번호를 발송해준다.

#비밀번호 변경
-> 비밀번호는 8자 이상 영문,숫자,특수문자 1글자 이상#

<여행후기>
#여행후기 작성
-> 제목,여행시작~끝날짜 , 경비, 내용 , 여행지를 입력받아야한다.
-> 각 여행지에는 여러 이미지와 글 하나를 작성할 수 있어야한다. -> PIN이 여러개 존재
-> 여행지는 kakao map api를 통해 검색하고 저장할 수 있다.

PIN -> record_id,pin_id,content, 장소 record
Image -> pin_id,image_id,origin,img

#여행후기 검색
-> 여행지 지역을 선택하면 해당 지역에 포함되는 여행후기가 나온다.
EX ) 경상북도 -> 안동 선택시 안동지역에 해당하는 여행후기가 나온다.
-> 클릭하면 자세한 여행후기 조회 가능

#여행후기 조회
-> 지도에 마커 표시가 되어있음
-> 마커에 마우스를 올리면 INFO창이 뜸
-> 제목,작성자,여행시작~끝날짜,경비,내용,여행지를 확인할 수 있다.
-> 여행지를 클릭하면 모달창이 뜨고 여행이미지를 슬라이드로 확인 할 수 있으며,글을 볼 수 있다.(카드형식?)
-> 댓글을 달수있다.(모달창??)


<여행계획/경로> -> 나만보기
->출발지 , 도착지 , 경유지를 입력해주면 최단경로를 지도에 그려준다.(drawing library , kakao mobility 경유지 길찾기)
->유저는 여러개의 여행계획(경로)를 가지고 여행계획(경로)는 여러개의 장소를 가진다. 각 장소는 순서가 정해져있다.
